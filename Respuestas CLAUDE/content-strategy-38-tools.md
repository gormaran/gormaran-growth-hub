## 1. Framework de Contenido — Estructura Universal por Post

Cada post sigue este esquema. Mismo molde, distinta herramienta:

```
URL:     /blog/[herramienta]-con-inteligencia-artificial
H1:      Cómo [hacer X] con IA: [herramienta] de Gormaran
Intro:   Problema → Coste del problema → Solución con IA (150 palabras)
H2:      ¿Qué es [herramienta] y para qué sirve?
H2:      El problema de hacer esto manualmente
H2:      Cómo funciona la herramienta (paso a paso con capturas)
H2:      Ejemplo real: input → output
H2:      Cuándo usar esta herramienta
H2:      Preguntas frecuentes (FAQ para featured snippets)
CTA:     Prueba [herramienta] gratis → link a trial
```

**Especificaciones técnicas:**
- Longitud: 1.200–1.800 palabras
- 1 keyword principal + 2 LSI en título/subtítulos
- 1 screenshot del output real de la herramienta
- Meta description: 155 caracteres con keyword + CTA
- Internal link: siempre a pricing + a 1 post relacionado

```markdown

**Herramienta:** [nombre]
**URL objetivo:** /blog/[slug]-con-inteligencia-artificial
**Keyword principal:** [keyword] (búsqueda mensual estimada: X)
**Keywords secundarias:** [LSI 1], [LSI 2]
**Buyer stage:** Awareness / Consideration / Decision / Implementation
**Audiencia primaria:** [freelancers / agencias / PyMEs / e-commerce / startups]
**Intención de búsqueda:** Informacional / Comercial / Transaccional

**Ángulo editorial:**
[Una frase que explique el giro del artículo]

**Pain point central:**
[El problema real que tiene el lector antes de encontrar la herramienta]

**Promesa del artículo:**
[Qué conseguirá el lector al terminar de leer]

**Ejemplo real a incluir:**
Input: [datos de ejemplo concretos]
Output esperado: [descripción del resultado]

**CTAs:**
- Principal: Prueba [herramienta] gratis → gormaran.io/register
- Secundario: Ver todos los planes → gormaran.io/pricing

**Internal links:**
- A: [post relacionado #X]
- A: /pricing

**Meta description (155 chars):**
[Keyword] + beneficio + CTA en menos de 155 caracteres.
```

## Ritmo de publicación sugerido

- 2 posts por semana = 38 posts en ~19 semanas (menos de 5 meses)
- Empezar siempre por los posts de prioridad 🔴 ALTA (posts #1–#10)
- Interlinkar cada post nuevo con al menos uno anterior de la misma categoría
